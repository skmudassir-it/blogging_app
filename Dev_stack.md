# Dev Stack

Tools and libraries used specifically for development workflow and environment.

## Development Tools
- **Code Editor**: VS Code / Cursor
- **Package Manager**: npm (Node Package Manager)
- **API Testing**: Postman, cURL
- **Version Control**: Git

## Development Dependencies
### Client
- `vite`: Fast localized dev server and bundler.
- `postcss`, `autoprefixer`: CSS processing.
- `eslint`: Code linting.

### Server
- `nodemon`: Auto-restarts server on file changes.

## Configuration Files
- `vite.config.js`: Frontend build config.
- `tailwind.config.js`: CSS framework config.
- `postcss.config.js`: CSS processor config.
- `.env`: Environment variables (API Keys, DB URIs).
