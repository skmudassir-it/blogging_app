# Tech Stack

## Frontend (Client)
- **Library**: React 19
- **Build Tool**: Vite
- **Styling**: Tailwind CSS v4
- **Routing**: React Router DOM v7
- **HTTP Client**: Axios
- **Icons**: Lucide React
- **Typography**: Google Fonts (Playfair Display, Lato)

## Backend (Server)
- **Runtime**: Node.js
- **Framework**: Express.js
- **Database**: MongoDB
- **ODM**: Mongoose
- **CORS**: cors middleware
- **Environment**: dotenv

## Deployment / Operations (Suggested)
- **Database**: MongoDB Atlas
- **Hosting**: Vercel (Client), Render/Heroku (Server)
